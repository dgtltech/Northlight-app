## What's new in 1.4

This release fixes how Northlight handles rotated photos.

### Fixed

- **Rotated photos now open correctly.** Photos carrying orientation metadata — most notably portrait shots from a phone (e.g. HEIC from iPhone) — were sized from their raw, unrotated pixel buffer. A portrait photo would open in a landscape-shaped window with the image floating between empty side margins, the zoom-to-fit was wrong, and the status bar showed the dimensions the wrong way around (e.g. `5712×4284` instead of `4284×5712`). Northlight now honors the orientation flag, so the window shape, fit, zoom percentage, and reported size all match what you actually see.
